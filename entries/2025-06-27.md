date: 2025-06-27

## New

- Register Interest Configuration — Create events that accept either standard bookings or registered interest bookings from the start. Set different booking styles for different events within the same collection, override booking styles via specific links or invites. Perfect for organizations displaying collections with mixed booking requirements.

- Unified Scanner Pipeline — Completely rebuilt scanning infrastructure providing consistent check-in experience across web scanner and CodeReadr, comprehensive check-in data storage for all scan types, and a new dedicated Scanners page for easier scanner management and configuration.

## Improved

- Added ability for organizations to assign portals to themselves — perfect for art fairs giving collectors their own portal access.

- Expanded portal notifications to include WhatsApp booking notifications.

- Enhanced portal settings page with clearer notification controls and contact information.

- Added custom payment descriptors in Stripe dashboard for better financial tracking.

- Improved banner upload feedback — now shows immediate visual confirmation when images are updated.

## Fixed

- Fixed banner upload loading states to properly reflect image changes.

- Resolved advanced event settings save issues.

- Corrected portal notification email configurations.

- Fixed event banner display inconsistencies.
