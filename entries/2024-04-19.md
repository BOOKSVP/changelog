## Improved
 

- API Rate Limiting protection has been improved. If your API requests 'Overheat' (hit the rate limits) constantly for over 60 minutes, API keys will temporarily be disabled for 24hrs. This added protection on DDOS attacks and ensures, should your key get in the hands of an attacker, your data is protected.
- Booking Link details no appear in the CSV download for all bookings.

## Fixed
 

- Turning On/Off Donations and Billing Address collection fix
- Turning On/Off Passwords for an event fix
