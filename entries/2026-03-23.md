---
title: "Custom Branded Tickets, Wallet Pass Templates & Navigation Updates"
date: 2026-03-23
---

[new]

- **Custom branded tickets** — Organisations can now customise ticket colours to match their event or company branding, replacing the default black-and-white design
- **Wallet pass templates** — Create and manage Apple Wallet and Google Wallet pass templates with custom colours, titles, and images through a dedicated settings page
- **What's New widget** — A lightweight in-app changelog now shows a badge when new updates are available, so you'll always know what's changed

[improved]

- **Guest List navigation** — Guest List now appears more prominently in your space navigation, making it quicker to find
- **Art Week Guide** — The ARTSVP Collective section has been renamed to Art Week Guide
- **Add to Outlook Calendar** — Calendar links in booking confirmation emails now work correctly on both mobile and desktop Outlook
- **Send Booking Details** — The send button is now disabled until a valid email address is entered, preventing failed sends

[fixed]

- Fixed an issue where new users accepting a membership invite could be incorrectly redirected to onboarding instead of joining the organisation
- Fixed a page error that could occur when visiting a mismatched subdomain URL (e.g. accessing one organisation's page on another's subdomain)
- Fixed a login error that could appear when returning to two-factor authentication after a session timeout
- Fixed an intermittent error on the Guest List page for bookings without an associated event
- Improved stability when guests update their details on the booking page
- Fixed an access issue with terminal settings for logged-out visitors
