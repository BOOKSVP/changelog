## New
 
Promo Codes Launched

We're excited to announce the launch of our new percentage-based promo codes! This feature allows users to apply a single discount code you've set to take a percentage off the final booking price (excludes donations, one promotional code per booking allowed).

Whether it's a special promotion or a reward for VIPs, you can configure multiple promo codes per event with varying percentages. Manage promo codes and track usage in the Edit section for your event under the Promo Codes tab.
